# SciSkillBench Benchmark Data

The benchmark questions and answers for SciSkillBench are available at Figshare:

**https://doi.org/10.6084/m9.figshare.30924998**

## Setup Instructions

1. Download the zip file from the Figshare link above
2. Extract the zip file with password: `sciskillbench`
3. Place the extracted `questions_and_answers/` folder in this directory

For additional setup requirements (MongoDB, XTB, LAMMPS, etc.), see [../deep_solver_benchmark/DEVELOPMENT.md](../deep_solver_benchmark/DEVELOPMENT.md).
