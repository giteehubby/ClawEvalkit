# Chapter 0: Introduction to Linux OS

## 1. What is Linux?
Linux is an open-source, Unix-like operating system kernel first released by Linus Torvalds in 1991. It serves as the core of various distributions (distros) like Ubuntu, Fedora, and Debian.

## 2. History and Evolution
- **1960s:** Unix development at Bell Labs.
- **1983:** Richard Stallman starts the GNU Project.
- **1991:** Linus Torvalds creates the Linux kernel.
- **Present:** Linux powers most of the internet, supercomputers, and Android devices.

## 3. Key Components of Linux
- **Kernel:** The heart of the OS, managing hardware and processes.
- **Shell:** The command-line interface (CLI) for interacting with the kernel (e.g., Bash, Zsh).
- **Desktop Environment:** The graphical user interface (GUI) (e.g., GNOME, KDE).
- **Applications:** Software like browsers, editors, and compilers.

## 4. Why Learn Linux?
- **Open Source:** Free to use, modify, and distribute.
- **Security:** Robust permission systems and community-driven patches.
- **Stability:** Known for high uptime and performance.
- **Career Growth:** Essential for DevOps, Cloud Computing, and Cybersecurity.

## 5. Basic File System Structure
- `/`: The root directory.
- `/bin`: Essential command binaries.
- `/etc`: Configuration files.
- `/home`: User personal files.
- `/var`: Variable data like logs.
