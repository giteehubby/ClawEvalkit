✅ A case where it helped ▾

Task: Fix the bug in calc.add so it returns the correct sum.

Without the skill, the agent couldn't generate a valid patch. With the skill, it correctly identified that return a - b should be return a + b.

--- a/calc.py +++ b/calc.py @@ -1,4 +1,4 @@ def add(a: float, b: float) -> float: - return a - b + return a + b
😐 A case where it didn't matter ▾

Task: Fix the slugify() function to use hyphen-separated slugs.

The skill is specific to arithmetic bugs, so it provided no relevant guidance for string manipulation tasks. The agent failed to generate a valid patch in both baseline and augmented runs.

Expected: "hello-world" Got: "helloworld" The skill's guidance about arithmetic patterns didn't apply to this string manipulation bug.
🔍 What we observed ▾

This skill is narrowly focused on arithmetic bugs in calc-style modules. It improved reliability on its target domain without negative side effects on unrelated tasks.

Recommendation: Consider pairing with complementary skills for broader coverage, or expanding the skill's patterns to cover more bug types.
