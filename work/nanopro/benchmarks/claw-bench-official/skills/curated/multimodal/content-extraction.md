# Content Extraction Skill

## Purpose
Extract structured data from various file formats including images, PDFs, and mixed-media documents.

## Capabilities
- Extract text content from structured documents
- Parse tabular data from formatted files
- Identify and extract metadata (dates, authors, titles)
- Convert between data representations (structured to unstructured and back)
- Handle multi-format input files with mixed content types

## Guidelines
- Preserve the original structure and hierarchy of extracted content
- Clearly mark uncertain or low-confidence extractions
- Handle missing or corrupted data gracefully with appropriate defaults
- Validate extracted data types (numbers, dates, etc.) against expected formats
- When extracting from complex layouts, maintain reading order
