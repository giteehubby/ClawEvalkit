# Multi-Domain Synthesis Skill

## Purpose
Coordinate information and actions across multiple domains to solve complex, multi-faceted tasks.

## Techniques

### Task Decomposition
- Break complex cross-domain tasks into single-domain subtasks
- Identify data dependencies between subtasks
- Plan execution order to minimize redundant work

### Data Integration
- Merge information from heterogeneous sources (logs, configs, databases, documents)
- Resolve schema differences when combining structured data
- Maintain provenance tracking for synthesized outputs

### Incident Analysis
- Correlate events across log files using timestamps
- Cross-reference configuration with observed behavior
- Build timelines from distributed event sources

### Report Generation
- Structure findings into actionable reports
- Include evidence and references for each finding
- Prioritize issues by severity and impact
