# Task: Add Type Annotations to Utility Functions

You are given a Python module `utils.py` in the workspace directory. It contains five utility functions that lack type annotations.

## Requirements

1. Add type annotations (parameter types and return types) to **all five** functions in `workspace/utils.py`.
2. Use standard Python typing where needed (e.g., `list`, `dict`, `Optional`, `Union`).
3. The file must still parse and execute without errors after your changes.
4. Do not change the logic or behavior of any function.

## Output

Save the updated file to `workspace/utils.py` (overwrite in place).
