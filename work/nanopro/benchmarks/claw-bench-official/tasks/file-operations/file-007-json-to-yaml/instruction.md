# Task: Convert JSON to YAML

You are given a JSON configuration file at `workspace/config.json`. Convert it to YAML format.

## Requirements

1. Read `workspace/config.json`.
2. Convert the JSON content to valid YAML.
3. Preserve all keys, values, nesting, and data types.
4. Write the result to `workspace/config.yaml`.

## Output

Save the YAML content to `workspace/config.yaml`.
