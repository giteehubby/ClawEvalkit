# Task: Find and Replace in Text File

You are given a text file at `workspace/input.txt`. Replace all occurrences of the word "foo" with "bar".

## Requirements

1. Read `workspace/input.txt`.
2. Replace every occurrence of `foo` with `bar` (case-sensitive, exact match of the substring).
3. Preserve the original line count and structure.
4. Write the result to `workspace/output.txt`.

## Output

Save the modified text to `workspace/output.txt`.
