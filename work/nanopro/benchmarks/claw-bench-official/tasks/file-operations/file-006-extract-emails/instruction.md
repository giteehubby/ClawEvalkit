# Task: Extract Emails from Text

You are given a text document at `workspace/document.txt` that contains email addresses embedded within prose. Extract all email addresses and write them to a new file.

## Requirements

1. Read `workspace/document.txt`.
2. Extract all email addresses from the text.
3. Sort the extracted emails alphabetically.
4. Write one email per line to `workspace/emails.txt`.

## Output

Save the sorted list of email addresses to `workspace/emails.txt`.
