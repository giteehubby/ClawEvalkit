# Task: Basic Statistics Computation

You are given a CSV file at `workspace/numbers.csv` with a column called `value`.

## Requirements

1. Read `workspace/numbers.csv`.
2. Compute the following statistics for the `value` column:
   - **mean**: arithmetic mean
   - **median**: middle value when sorted
   - **mode**: most frequently occurring value
   - **std_dev**: population standard deviation
3. Write the results to `workspace/stats.json` as a JSON object with keys: `mean`, `median`, `mode`, `std_dev`.
4. All floating-point values should be rounded to 2 decimal places.

## Output

Save the JSON result to `workspace/stats.json`.
