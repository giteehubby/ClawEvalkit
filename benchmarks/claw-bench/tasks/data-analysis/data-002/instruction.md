# Task: Sort and Filter CSV Data

You are given a CSV file at `workspace/people.csv` with columns: `name`, `age`, `city`, `score`.

## Requirements

1. Read `workspace/people.csv`.
2. Filter rows where `age > 25`.
3. Sort the filtered rows alphabetically by `name`.
4. Preserve the header row.
5. Write the result to `workspace/filtered.csv`.

## Output

Save the filtered and sorted CSV to `workspace/filtered.csv`.
