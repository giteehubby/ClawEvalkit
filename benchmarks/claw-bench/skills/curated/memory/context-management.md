# Context Management Skill

## Purpose
Maintain and recall information across multi-step task execution.

## Techniques

### Working Memory
- Keep a scratchpad of key facts extracted from inputs
- Reference earlier instructions explicitly when producing output
- Track state across sequential steps

### Instruction Parsing
- Number and categorize all instructions before execution
- Identify dependencies between steps
- Flag contradictions or ambiguities early

### Cross-Reference
- When multiple data sources are provided, identify shared keys
- Build a mental model linking information across files
- Verify completeness — ensure all sources are represented in output

### Disambiguation
- When requirements are vague, state assumptions explicitly
- Prefer the most recent instruction when contradictions exist
- Document interpretation choices in output
