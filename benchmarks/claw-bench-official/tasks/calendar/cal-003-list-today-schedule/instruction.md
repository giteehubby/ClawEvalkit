# Task: List Today's Schedule

You are given a calendar file at `workspace/calendar.json` containing events across multiple days.

Extract all events scheduled for **2026-03-15** and write them to `workspace/today.json`, sorted by `start_time` in ascending order.

## Output

Write the filtered events to `workspace/today.json` as a JSON object with a `"events"` key containing the sorted array of matching events. Each event must retain all its original fields.
