# Task: Fix Syntax Errors in Python Module

You are given a Python module `broken.py` in the workspace directory. It contains **three syntax errors** that prevent it from being imported or executed.

## Requirements

1. Identify and fix all three syntax errors in `workspace/broken.py`.
2. The file must parse without any `SyntaxError`.
3. All three functions (`greet`, `compute_average`, `build_report`) must return correct values.
4. Do not change the intended logic or behavior of the functions.

## Output

Save the fixed file to `workspace/broken.py` (overwrite in place).
