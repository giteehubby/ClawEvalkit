# Task: Refactor For-Loops to List Comprehensions

You are given `workspace/process.py` containing four functions that each use a for-loop to build a list. Refactor each function to use a list comprehension instead.

## Requirements

1. Replace the for-loop pattern in each of the four functions (`get_even_numbers`, `get_uppercased`, `get_lengths`, `filter_positive`) with a list comprehension.
2. Each function must produce the **exact same output** as the original.
3. Do not add or remove any functions.
4. Do not use explicit for-loops for list building in those four functions.

## Output

Save the updated file to `workspace/process.py` (overwrite in place).
