# Task: Implement a Palindrome Checker

Write a Python function `is_palindrome(s)` in `workspace/palindrome.py` that determines whether a given string is a palindrome.

## Requirements

1. The function should accept a single string argument `s`.
2. Return `True` if the string reads the same forwards and backwards, `False` otherwise.
3. The check should be case-insensitive (e.g., "Racecar" is a palindrome).
4. Ignore non-alphanumeric characters (e.g., "A man, a plan, a canal: Panama" is a palindrome).
5. An empty string should be considered a palindrome.
6. A single character should be considered a palindrome.

## Output

Save the file to `workspace/palindrome.py`.
