# Task: Deduplicate Lines

You are given a text file at `workspace/data.txt` that contains some duplicate lines. Remove the duplicates while preserving the order of first occurrences.

## Requirements

1. Read `workspace/data.txt`.
2. Remove all duplicate lines, keeping only the first occurrence of each line.
3. Preserve the original order of the remaining lines.
4. Write the result to `workspace/unique.txt`.

## Output

Save the deduplicated lines to `workspace/unique.txt`.
