Memory Domain — All Models Score 50.0
The memory domain consists of 5 tasks: constraint-accumulation, context-retention, interleaved-projects, memory-organization, recall-distraction. All 13 model variants — including exp87_step170, exp87_ratio5pct_step1320, exp87_step1360, long_rl, seed18 — score exactly 50.0. This is a pure C-type (state management) failure: nanobot's AGENTS.md/SOUL.md initialization mechanism does not support cross-session memory and constraint tracking required by these tasks. The 50.0 likely comes from random correct answers or minimal task completion.
⚠ Infrastructure-bound: Any training-free recipe or SFT approach will be ineffective unless paired with infrastructure-level changes to the nanobot session management system.
