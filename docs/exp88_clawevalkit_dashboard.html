API Routing Logic (NanoBotAgent._call_llm)

requested_model = model or self.model is_openrouter = "openrouter" in api_url # OpenRouter proxy is_anthropic = "anthropic" in api_url # MiniMax/GLM Anthropic-compat is_azure = "azure" in api_url # Azure proxy if is_openrouter: prefix = "openrouter/" # litellm openrouter/{model} elif is_anthropic: prefix = "anthropic/" # litellm anthropic/{model} else: prefix = "openai/" # litellm openai/{model} (ARK, etc.)

Add a New Model (5 Lines YAML)

# configs/models/my_model.yaml my-model: name: My Model api_url: https://api.example.com/v1 api_key_env: MY_API_KEY model: my-model-name provider: custom
# Then run: python3 run.py --model my-model --bench tribe
